# Xeno CRM

## Features

- Customer Management
- Order Management
- AI Copilot
- Campaign Management
- Analytics Dashboard
- Campaign History
- Channel Service
- Receipt Callback

## Tech Stack

- React
- Axios
- React Router

Backend:
- FastAPI
- SQLAlchemy

Database:
- Neon PostgreSQL
